Add a Spellcheck Feature to Your NodeJS Project

7:50 AM October 24th, 2017

This month is Hacktober, and it's a great opportunity to add something to an open source project! My first pull request (link) was to SendGrid's docs repo, adding a spellchecking feature. This feature is so important for large-scale projects where many people contribute to markdown and html pages, and it's hard to keep track of all of the changes! Even on a personal scale, this feature is great for projects like this blog.

Setup

Even if you're not familiar with NodeJS or NPM, this shouldn't be too hard. NodeJS is amazing out-of-the-box. I'm not going to explain how to setup NodeJS and NPM, but you can learn how to do so (here).

If you're starting from a brand new directory, initalize the folder as a typical NodeJS project.
# setup # init repo e.g. "documentation" > mkdir documentation && cd documentation # init npm and package.json > npm init # install dependencies > npm install # test init > npm start
